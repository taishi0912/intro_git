ダークダンジョン (Unity3D)

Unityを使用して開発したアクションRPGゲーム

ダークダンジョンは、Unity3Dを使用して開発したアクションRPGゲームです。プレイヤーは勇者となり、様々なダンジョンを探索しながら、モンスターと戦い、宝箱を探す冒険を楽しめます。このゲームの主な特徴は以下の通りです：

  • Unityエンジンを使用した3Dゲーム開発
  • C#によるゲームロジックの実装
  • NavMeshを活用した敵AIの実装
  • パーティクルシステムを用いた視覚効果の強化
  • 複数のシーンを用いたゲームフロー設計
  • オーディオシステムによるサウンド実装

技術的な実装の詳細：

  1. Unityエンジンを活用したゲーム開発
    • 3Dモデリングとアニメーションの統合
    • シーン管理によるゲームの構造化
  2. C#スクリプティングによるゲームロジックの実装
    • オブジェクト指向プログラミングの活用
    • コルーチンを用いた非同期処理の実現
  3. NavMeshを使用した敵AIの実装
    • 敵キャラクターの自動経路探索
    • プレイヤーの追跡と攻撃行動のプログラミング
  4. パーティクルシステムによる視覚効果
    • 魔法や攻撃エフェクトの作成
    • 環境演出のためのパーティクル使用
  5. 複数シーンを用いたゲームフロー設計
    • タイトル画面、メインゲーム、ゲームオーバー画面の遷移
    • シーン間でのデータ受け渡しの実装
  6. オーディオシステムの実装
    • BGMと効果音の管理
    • 3D音響効果の活用
  7. ユーザーインターフェースの設計と実装
    • HPバーやスコア表示などのUI要素の作成
    • Canvas使用によるレスポンシブUIの実現
  8. キャラクター制御システムの実装
    • キーボード入力による移動と攻撃の制御
    • カメラワークの最適化
  9. コリジョン検出とダメージシステム
    • 衝突判定による攻撃とダメージの計算
    • ヒットボックスの調整
  10. ゲームバランスの調整
    • 敵の強さや出現頻度の調整
    • アイテムやスコアシステムの設計

このプロジェクトでは、Unityを用いた3Dゲーム開発の基本的なスキルに加え、ゲームデザインとプログラミングの融合、ユーザーエクスペリエンスを考慮したゲーム設計能力を示しています。また、複雑なゲームシステムを構築し、視覚的にも聴覚的にも魅力的なゲーム体験を提供する能力も実証しています。

Ballon (Ruby)

高校2年生秋から3年生にかけて開発した悩み相談Webサービス

Ballonは、中高生向けの悩み相談ウェブサービスです。このアプリケーションは、悩みの共有とコミュニティサポートを組み合わせることで、若者の精神的健康とピアサポートを促進します。主な特徴は以下の通りです：

  • Sinatraを使用したバックエンド開発
  • Active Recordを用いたデータベース管理
  • CSSを活用したレスポンシブデザイン
  • ユーザー認証システムの実装
  • 悩みの投稿・共有機能
  • 回答機能によるコミュニティサポート
  • いいね機能によるユーザー間のインタラクション

技術的な実装の詳細：

  1. Sinatraフレームワークを使用したサーバーサイド開発
    • ルーティング、コントローラー、ビューの実装
    • セッション管理によるユーザー認証の実装
  2. Active Recordを用いたORMによるデータベース操作
    • ユーザー、悩み投稿、回答、いいねのモデル定義
    • リレーションシップの設定による効率的なデータ管理
  3. CSSを活用したレスポンシブデザインの実装
    • フレックスボックスレイアウトを用いた柔軟なデザイン
    • ボトムナビゲーションによるモバイルフレンドリーな UI
  4. ユーザー認証システムの実装
    • BCryptを使用したパスワードのハッシュ化
    • サインアップ、サインイン、サインアウト機能の実装
  5. 悩み投稿機能の実装
    • ユーザーごとの悩み投稿・表示機能
    • 悩み詳細ページの実装
  6. 回答機能の実装
    • 悩みに対する回答の作成・表示機能
    • ユーザーと回答の関連付け
  7. いいね機能の実装
    • 回答に対するいいねの追加機能
    • ユーザーといいねの関連付け

このプロジェクトでは、Webアプリケーション開発の基本的なスキルに加え、ユーザー間のインタラクションを促進するための機能設計能力を示しています。また、若者の悩み相談という重要な問題に対して、匿名性を保ちながらコミュニティサポートを提供する創造的なソリューションを実現しています。

Ballonは、中高生の精神的健康サポートとコミュニティ形成の両立を目指す、現代の若者のニーズに応えたWebアプリケーションです。安全でサポーティブな環境を提供することで、若者たちが互いに助け合い、成長できるプラットフォームを実現しています。

Daily Beat(Python)

大学入学後に開発した音楽共有SNS

Daily Beatは、音楽を通じて人々をつなげる新しいSNSプラットフォームです。ユーザーは毎日1曲を共有し、音楽を通じて自分の感情や経験を表現できます。このアプリケーションの主な特徴は以下の通りです：

  • Flaskを使用したバックエンド開発
  • SQLAlchemyを用いたデータベース管理
  • JavaScriptとCSSを活用したレスポンシブなユーザーインターフェース
  • 日々の曲投稿機能による継続的なエンゲージメント
  • フォロー機能によるソーシャルネットワーキング
  • アルバムアートの自動取得機能
  • バックグラウンドジョブによる定期的なデータ更新

技術的な実装の詳細：

  1. Flask フレームワークを使用したサーバーサイド開発
    • ルーティング、ビュー関数、テンプレートレンダリングの実装
    • Flask-Loginを使用したユーザー認証システムの構築
  2. SQLAlchemyを用いたORMによるデータベース操作
    • ユーザー、投稿、フォロー関係等のモデル定義
    • Flask-Migrateを使用したデータベースマイグレーション管理
  3. JavaScriptを活用したフロントエンドの対話性向上
    • アルバムアートの非同期取得によるユーザー体験の向上
    • 動的なコンテンツ更新によるシームレスな操作感の実現
  4. CSSによるレスポンシブデザインの実装
    • フレックスボックスレイアウトを用いた適応型デザイン
    • メディアクエリによる様々なデバイスサイズへの対応
  5. 日々の曲投稿機能の実装
    • WTFormsを使用したフォームバリデーション
    • ユーザーごとの1日1投稿制限の実装
  6. フォロー機能の実装
    • 多対多リレーションシップを用いたフォロー関係の管理
    • フォロー中ユーザーの投稿フィルタリング機能
  7. アルバムアート自動取得機能の実装
    • Spotify APIを利用した楽曲情報の取得
    • 取得したデータのキャッシュによるパフォーマンス最適化
  8. バックグラウンドジョブの実装
    • APSchedulerを使用した定期的なデータ更新処理
    • 毎日の投稿リセット機能の自動化

このプロジェクトでは、Webアプリケーション開発の基本的なスキルに加え、ソーシャルネットワーキングプラットフォームの設計と実装能力を示しています。また、音楽共有という特定のニーズに対して、技術を活用して創造的なソリューションを提供する能力も実証しています。Daily Beatは、ユーザー体験を重視しながら、効率的なバックエンド処理と魅力的なフロントエンドデザインを組み合わせた総合的なWebアプリケーション開発のスキルセットを示す優れた例となっています。

  • タイムラインによる選択の可視化
  • 未来の自分との対話機能の実装

技術的な実装の詳細：

  1. Flask フレームワークを使用したサーバーサイド開発
    • ルーティング、ビュー関数、テンプレートレンダリングの実装
    • セッション管理によるユーザー認証の実装

    • ユーザー情報、キャリアパス、タイムライン等のモデル定義

  3. JavaScriptを活用したフロントエンドの対話性向上
    • 非同期通信によるシームレスなユーザー体験の実現

  5. ゲーミフィケーション要素の実装
    • タイムクリスタルシステムによるユーザーエンゲージメントの向上
    • 重要な分岐点の設定による戦略的な意思決定の促進
  6. 適性診断アルゴリズムの実装
    • ユーザーの回答に基づく業界適性の算出

    • 重要な選択のハイライト表示
  8. 未来の自分との対話機能
    • 自己分析結果の表示
    • チャットインターフェースによる対話シミュレーション
